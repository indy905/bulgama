# 복선 및 유효 포인트 추적 (Foreshadowing & Plot Point Tracker)

이 파일은 작품 내에 심어진 복선(Foreshadowing)과 향후 회수해야 할 설정들을 추적하여 서사의 완결성을 높이는 데 사용됩니다.

## 1. 활성 복선 (Active Foreshadowing)
*현재 심어졌으나 아직 회수되지 않은 복선들입니다.*

- **[착각] 제국 정찰대의 오해:** 온돌 공사를 '지옥의 가마솥(멸망 병기)' 주조로, 성검 파쇄 소음을 '용사 고문 소리'로 착각하여 2차 대규모 토벌대를 요청함. (3막 황제 대군 침공의 도화선)
- **[착각] 전술 기동복(찜질복)의 위협:** 6화에서 배포된 황토색 찜질복을 제국 본국에서는 "마력 억제 천으로 기사들의 마법 저항력을 0으로 만든 뒤, 마수의 가죽으로 피부를 대체하는 기괴한 강화 수술"로 오해할 소지가 있음.
- **[착각] 대욕탕 고문실:** 7화에서 기사들이 지옥염으로 끓인 온탕에 들어간 사건이 훗날 제국 본국에 "마왕이 기사들을 펄펄 끓는 솥단지에 삶아 세뇌시킨다"라는 끔찍한 오해로 와전될 소지가 있음.
- **[설정] 제국의 난방 독점 비리:** 카엘이 추방된 근본 원인. 제국 고위층과 '백색 빙하기'의 연관성 암시.
- **[아이템] 아카식 톱니바퀴:** 제국 시스템의 핵심 유물. 단순히 등급을 매기는 것 이상의 숨겨진 기능 존재 가능성.
- **[인물] 서리 신 (Frost God):** 빙하기의 원흉으로 지목됨. 5막에서 최종 보스로 등장 예정.
- **[캐릭터] 진우의 이세계 소환:** 제국이 왜 하필 '한국인'을 소환했능지에 대한 잠재적 이유.

## 2. 회수된 복선 (Resolved Foreshadowing)
*성공적으로 서사 내에서 회수된 복선들입니다.*

- **[아이템] 드레스 코드 통일:** 기사들의 무질서한 노출(시각적 테러) 문제를 샌드웜 가죽과 마력 억제 천을 결합한 '찜질복'으로 해결. (6화 완료)

## 3. 설정 충돌 주의보 (Consistency Alerts)
*작성 시 주의가 필요한 설정들입니다.*

- **마나-체온 등가교환:** 마나 사용 시 체온 저하 묘사가 누락되지 않도록 주의.
- **양머리 수건의 오해:** 제국 인물들이 이를 마왕의 뿔로 계속 착각하는 연출 유지.

## 4. 향후 활용 아이디어 (Future Ideas)
- 찜질방 쿠폰 시스템을 통한 제국 기사들의 충성도 테스트.
- 마석을 활용한 최첨단 안마의자(마법 공학 결합) 도입.
